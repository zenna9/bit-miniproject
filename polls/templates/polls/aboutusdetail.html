따뜻한 밥을 제 때에
따박 따박 챙겨먹는다.

'따밥따밥'은 사회적 취약 계층을 대상으로 언제나 따뜻한 밥 한끼를 먹을수 있도록 '무료급식소 데이터', '선한영향력 가게' 등 여러 데이터를 출력해주는 웹 어플리케이션입니다.

'따밥따밥'의 주요 기능 보러가기

대한민국 모두가 밥을 굶지 않는 그날까지 따밥따밥이 함께합니다.

따밥따밥의 주요 기능

'선한 영향력 가게' 찾기

결식아동에게 식사를 지원하는 '선한영향력 가게'의 주소, 음식 구분에 따라 지도에 보여줍니다.

지역별 데이터 추출

지역 별, 식당 종류별, 운영시간 등 여러 조건을 설정해 원하는 정보를 볼 수 있습니다.

무료급식소 데이터

저소득층, 노약자, 장애인 등 사회적 취약 계층을 대상으로 한 무료급식소 현황을 보여줍니다.

기존 선한영향력 가게 찾기 사이트와는 어떤 차별점이 있는지 궁금하신가요?

'따밥따밥' 세부기능 알아보기

기존 '선한영향력 가게' 사이트와의 차별점

구글 지도와 연동

해당 식당의 상세정보를 구글 지도에서부터 추출해서 보여주고, 해당 위치를 구글 지도에 자동 검색하여 연결해줍니다.

지도 상에서 가게정보 열람

좌표를 클릭하면 페이지 이동 없이도 지도 상에서 간략한 가게정보를 한눈에 확인 할 수 있습니다.

현재 영업중인 식당 목록

현재 시간을 기준으로 영업중인 식당과 영업 종료 식당데이터를 선택해서 출력이 가능합니다.

지역별 가게정보 추출

지역별 음식점 밀집정도를 숫자와 매핑 시각화로 표시하여서 지역별 음식점 수를 확인 할 수 있습니다.

다양한 메뉴

지도 페이지에서 여러가지 기능 페이지로 이동할 수 있도록 메뉴 바를 생성해 줍니다.

사용자의 실시간 위치 표시

웹을 사용하고있는 현재 사용자의 위치를 실시간으로 조회하여 지도상에 현재 위치를 마커로 표시해줍니다.
